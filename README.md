# Ibasho（SNS）

## 概要
Ibashoは現在構築中の簡易的なSNSサービスです。
C#×Blazor×EFCoreの技術スタックにて開発しているWebアプリケーションであり、サービスのローンチ目的ではなく学習目的です。
基本的なCRUDの動作から、リアルタイムな動作までBlazorを駆使して構築していきます。

## 技術スタック
- 言語: C#
- フレームワーク: .NET Blazor
- ORM: EFCore
- DB: PostgreSQL
- CSS: TailWindCSS
- 開発環境: VisualStudio2022 VSCode
